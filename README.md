# 法学论文写作 Skill

> 基于何海波《法学论文写作》（北京大学出版社）方法体系封装的 Claude Code Skill

## 简介

本仓库是一个 [Claude Code](https://claude.ai) 可用的 Skill，将何海波教授《法学论文写作》一书的完整方法论体系封装为可交互、可操作的步骤指南。覆盖法学论文从选题到发表的全流程。

### 关于原书

《法学论文写作》由何海波教授（清华大学法学院）撰写，北京大学出版社出版，是中文法学界公认的论文写作经典教材。全书共七讲，系统涵盖了法学论文写作的各个环节。

## 安装

### 前提条件

- 安装 [Claude Code](https://claude.ai)
- 拥有一个 Claude Code 项目

### 安装步骤

**方式一：直接克隆到本地**

```bash
git clone https://github.com/Cangechina/legal-writing-skills.git
cd legal-writing-skills
# 在项目中使用 /legal-writing 调用 skill
```

**方式二：将 Skill 文件复制到你的项目中**

```bash
# 在你的 Claude Code 项目目录下执行
mkdir -p .claude/skills/legal-writing
cp /path/to/legal-writing-skills/.claude/skills/legal-writing/SKILL.md .claude/skills/legal-writing/
```

**方式三：子模块方式**

```bash
git submodule add https://github.com/Cangechina/legal-writing-skills.git .claude/skills/legal-writing
```

### 验证安装

启动 Claude Code，输入 `/legal-writing`，如果能看到本 skill 的加载提示，则安装成功。

## Skill 内容概览

本 Skill 将原书七讲方法拆解为七个可操作阶段：

### 第一阶段：选题指导

- 判断选题是否属于法学论文
- 从阅读、争论、反常现象中发现真问题
- "一句话测试"检验核心命题
- 五维可行性检验（问题意识 / 创新可能 / 知识结构 / 资料可得 / 时间考量）
- 先期调研与论题调整

### 第二阶段：文献检索

- 穷尽相关文献的四项检验标准
- 三方法交替检索：地毯式排查 + 顺藤摸瓜 + 补充检索
- 关键词迭代技巧
- 文献质量判断与整理规范

### 第三阶段：调查研究

- 四种调查方法：现场观察 / 深度访谈 / 问卷调查 / 文献分析
- 各方法的操作步骤与注意事项
- 数据可信度评估

### 第四阶段：论证构建

- 六种论证方法：状况描述 / 属性分析 / 原因分析 / 功能分析 / 法律解释 / 政策建议
- 法律解释方法次序：文义 → 体系 → 历史 → 目的
- 成文法源与不成文法源的运用
- 因果关系分析的陷阱（共生现象 / 多因一果）

### 第五阶段：论文各部分写作

- 标题的命名规则与类型选择
- 导论四要素：宣明主题 / 综述文献 / 基本立场 / 文章结构
- 结论三部分：概括观点 / 延伸意义 / 交代局限
- 引注规范与格式
- 摘要与关键词的写作规范
- 目录结构设计（并列式 / 推进式 / 混合式）

### 第六阶段：行文写作

- 措词：文字干净 / 术语规范 / 说话有分寸 / 间接引用
- 句段：句式技巧 / 段落规范（每页必有分段）
- 章节结构设计
- 排版规范（字号 / 行距 / 标题层次 / 脚注）

### 第七阶段：学术伦理

- 调查伦理：知情 / 同意 / 无害
- 三条红线：抄袭 / 剽窃 / 伪造注释
- 投稿伦理：一稿一投 / 重复发表 / 转载

### 速查表

| 阶段 | 核心任务 | 关键产出 | 参考耗时 |
|------|---------|---------|---------|
| 选题 | 发现→提炼→检验→确定 | 核心命题（一句话） | 1—3个月 |
| 文献 | 地毯检索→延伸阅读→整理 | 文献综述 | 1—2个月 |
| 调查 | 观察/访谈/问卷/文献分析 | 一手数据 | 1—3个月 |
| 论证 | 描述→分析→解释→建议 | 论文主体 | 2—6个月 |
| 成文 | 标题→导论→正文→结论→摘要 | 完整初稿 | 1—3个月 |
| 修改 | 措词→句段→结构→排版 | 终稿 | 2—4周 |
