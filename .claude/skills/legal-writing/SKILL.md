# 法学论文写作指导

基于何海波《法学论文写作》（北京大学出版社）方法体系。覆盖法学论文从选题到发表的全流程。

---

## 使用说明

**触发场景：**
- 用户说"帮我选题""我不知道写什么"
- 用户说"文献怎么查""找不到文献"
- 用户说"论证方法""法律解释怎么做"
- 用户说"论文结构""导论怎么写""摘要怎么写"
- 用户说"行文技巧""语言规范""排版"
- 用户说"学术伦理""抄袭""引用规范"
- 用户说"写一篇法学论文""法学论文写作方法"

**响应方式：**
根据用户所处的写作阶段，提供对应的操作步骤清单。如果用户未指明阶段，先询问当前进度（选题/文献/调查/论证/成文/修改），再提供对应指导。

**参数传递：**
`$ARGUMENTS` 包含用户的具体问题或写作阶段关键词。

## 第一阶段：选题指导

### 操作流程

**步骤1：判断你的选题是否属于法学论文**
- 确认研究对象是法律现象（凡是涉及人类社会组织规则的都可以）
- 确定研究视角：法官视角（法律解释）| 立法者视角（制度完善）| 决策者视角（最优决策）| 观察者视角（社会描述）
- 区分体裁：论文（有核心命题）| 综述（述而不作）| 评论（以评为主）
- 区分类型：规范研究（"应当怎么样"）| 实证研究（"事实怎么样"）

**步骤2：发现问题**
- 精读现有文献，发现前人未触及的领域
- 从常识的矛盾中发现问题——流行的观点是否有例外？
- 从持久争论中发掘深层理论分歧
- 从实际与制度的落差中找到真问题

**步骤3：用"一句话测试"检验选题**
```
问自己：我能用一句话概括我要论证的核心观点吗？
如果不能 → 选题尚未成型，继续提炼
如果能   → 进入下一步检验
```

**步骤4：检验选题的可行性**

| 检验维度 | 操作标准 |
|---------|---------|
| 问题意识 | 这是有代表性的真问题吗？避开几十年碰不上的假问题 |
| 创新可能 | 查文献确认前人未充分讨论；是新问题/新见解/新材料/新方法？ |
| 知识结构 | 基础知识储备是否足够？临时补来不及的就放弃 |
| 资料可得 | 公开文献是否足够？需实地调查的是否能获取？ |
| 时间考量 | 学位论文要考虑一年两载能否完成 |

**步骤5：先期调研与调整**
1. 用CNKI检索该主题研究到什么程度
2. 拟写初步提纲，列出参考文献
3. 与人讨论（开题时听取意见）
4. 调整：偏大则限缩拆分，偏小则深入扩展，方向不对则转移

**常见错误警示：**
- ❌ "××若干问题研究"——多中心，不聚焦
- ❌ 教科书式写法——概念→性质→分类→意义，没有核心命题
- ❌ 题目过大——如"法治政府建设研究"
- ❌ 冥想式问题——几十年碰不上一回的假问题

## 第二阶段：文献检索

### 操作流程

**步骤1：穷尽相关文献的四项检验**

- [ ] 是否检索了法学论文、专著和教材？（学术圈的基本门槛）
- [ ] 是否检索了立法文本、判例和时事报道？（法学论文的必要素材）
- [ ] 是否检索了非法学的人文社科文献？（拓宽视野）
- [ ] 资料新旧是否得当？（最新发展 + 历史溯源）

**步骤2：三方法交替检索**

**方法A——地毯式排查：**
1. 选择数据库（CNKI / 北大法宝 / Westlaw / Heinonline）
2. 标题检索 → 主题检索 → 关键词检索 → 全文检索，由窄到宽
3. 检索词迭代技巧：
   - 尝试不同表述（"合同"也搜"契约""协议"）
   - 注意不同时期用语差异（"听证"早期叫"申辩"）
   - 注意不同地区用语差异（大陆"行政行为"= 台湾"行政处分"）
   - 从阅读中发现新关键词 → 重新排查 → 循环

**方法B——顺藤摸瓜：**
1. 找到一篇高质量论文，沿其参考文献不断追溯
2. 查看引用某篇重要文献的后世文献
3. 注意数据库提供的"同时被引"文献

**方法C——补充检索：**
- 从延伸阅读中发现遗漏的关键词 → 用新关键词重新排查
- 经过多个来回，直到对相关文献了如指掌

**步骤3：判断文献质量**
- 先借助外部指标：期刊权威性、被引频次、作者声望
- 再阅读判断：思想的原创性、论证的严谨性、影响力
- 原则：一手资料是金，二手资料是铜，三手资料是垃圾

**步骤4：文献整理**
- 每篇论文建一个文件夹
- 整理文献目录，必要时做摘要
- 注意备份

## 第三阶段：调查研究

> 实证研究需要；纯规范研究可跳过。

### 四种调查方法

**A. 现场观察：**
1. 明确观察目标、方式和记录项目
2. 选择不介入或参与式观察
3. 多次观察，单次不可靠

**B. 深度访谈：**
1. 找了解情况的人作为访谈对象
2. 设计访谈提纲
3. 建立信任（"找熟人"是好办法）
4. 应当让访谈对象知情并同意
5. 警惕被访谈者的形象塑造偏差

**C. 问卷调查：**
1. 问卷设计：格式化答案便于统计；问题要清楚、简单、中肯；敏感信息放最后；先预调查
2. 抽样：概率抽样最可靠；注意应答率
3. 评估：排除无效问卷；考虑"沉默群体"造成的取样偏差

**D. 文献分析：**
1. 确定样本范围——样本是否具有代表性？取样标准要操作化
2. 定量（大样本）或定性（小样本）分析
3. 检验标准：他人重复统计应得基本一致的结果
4. 数据呈现：图表更直观，数字精确到能说明情况即可

## 第四阶段：论证构建

### 六种论证方法

**1. 状况描述**

| 方法 | 操作要点 |
|------|---------|
| 举例说明 | 真实典型；支撑同一观点的事例不超过3个；叙述精炼 |
| 统计数据 | 核查可靠性；用最新数据；确保与主题相关（防数据陷阱） |
| 类比说明 | 只能辅助，注意局限性 |

**2. 属性分析**
- 搜集具有"家族相似"的事例，归纳共同形态
- 与相关概念对比，厘清界限
- 概念使用：抓本质特征、保持同一性、尊重学界既有理解
- 理想类型（模式）建构：适用于宏观抽象分析

**3. 原因分析**
- 寻找有意义的联系（从法律秩序维护或改进出发）
- 聚焦一个原因，不要面面俱到
- 警惕"共生现象"（A和B同时出现是因为C）
- 注意"多因一果"和"原因背后的原因"

**4. 功能分析**
- 显功能（有意安排的后果）vs 潜功能（无意图的后果）
- 评估：预定功能发挥得怎么样？
- 机理：为何有此功能？为何失效？

**5. 法律解释（规范研究核心）**
- 确定法律渊源：成文法源（宪法/法律/法规/规章）vs 不成文法源（学说/先例/公共道德/比较法）
- 成文法源论证：法律条文是"论据之王"——注意条文含义的明确性、效力、与论证主题的相关性
- 法律解释方法次序：文义解释 → 体系解释 → 历史解释 → 目的解释
- 合理性解释：利益衡量、社会学解释、经济学分析

**6. 政策建议**
- 建立在充分论证基础上
- 学者更适合分析条件和后果，而非简单拿出对策

## 第五阶段：论文各部分写作

### 5.1 标题

**检验清单：**
- [ ] 名实相符——讨论范围与正文相当
- [ ] 新颖有力——避免"法治视野下的××"等陈词
- [ ] 简洁顺畅——能省一字就省一字
- [ ] 避免三层结构（标题: 破折号:）
- [ ] 能用一句话说清楚就不用副标题

**标题类型选择：**
- 表明论题：如"行政诉讼撤诉考"
- 表明命题：如"软法亦法"
- 设问式：如"烟草是药吗？"

### 5.2 导论（四要素）

1. **宣明主题**——讨论什么问题？为什么重要？
2. **综述文献**——前人研究过什么？还有什么不足？
3. **基本立场和方法**——核心命题是什么？
4. **文章结构**——篇章安排

技巧：开门见山，可先用案例/轶事/数据做引子；不要客套。

### 5.3 结论（三部分）

1. 概括文章观点（提炼升华，非简单重复）
2. 延伸论证的意义（理论贡献/实践启示）
3. 交代研究的局限（方法不足/适用范围限制）

### 5.4 引注规范

**基本规则：**
- 间接引用优于直接引用——用自己的话归纳
- 原文引用只限于简短摘录重要文字
- 转引必须注明"转引自"
- 绝对禁止抄袭和伪造注释

**格式示例：**
- 期刊论文：作者《文章名》，《刊物名》年份期号
- 专著：作者《书名》，出版社版本，页码
- 译著：[国别]作者《书名》，译者译，出版社版本，页码

### 5.5 摘要和关键词

**摘要写作规则：**
- 200—300字，单段落，陈述句
- 不使用"本文"一词
- 逻辑链：问题 → 分析 → 对策
- 避免第一人称

**关键词规则：**
- 4—5个实体性名词短语
- 第一个关键词为核心概念
- 涵盖"概念层—制度层—理论层"梯度

### 5.6 目录结构

- 论文分2—3个层次
- 每个层次至少3个部分，一般不超过7—8个
- 结构逻辑：并列式（各部分独立并列）| 推进式（层层深入）| 混合式
- 检验：做成目录后整体审视逻辑是否清晰

## 第六阶段：行文写作

### 6.1 措词

**操作清单：**
- [ ] 文字干净：去掉废词（不必要的"的""但是…却"等）
- [ ] 正文少夹英文：除非必要（无统一译法的术语）
- [ ] 少用解释性脚注：能在正文说就在正文说
- [ ] 术语规范：同一概念用相同语词；同一语词含义前后一致
- [ ] 说话有分寸：避免全称判断（"都""所有"），慎用最高级
- [ ] 学会间接引用——用自己的话归纳，减少直接引用

### 6.2 句段

**句式：**
- 慎用百字以上超长句
- 合理使用连接词（首先/其次、但是、因此、总之）标示逻辑
- 少用插入语和破折号

**段落：**
- 每段一个中心观点，主题句放段首
- 每页纸必有分段（约二三百字即可分段）
- 段落字数200—400字为宜
- 长短句兼有

### 6.3 章节结构

- 并列式：各部分独立并列
- 推进式：步步为营层层深入
- 混合式：两者组合

检验方法：做成目录后整体审视。

### 6.4 排版规范

| 元素 | 规范 |
|------|------|
| 正文 | 宋体小四（12号），1.5倍行距，段首缩进两字 |
| 标题层次 | 用"一、（一）、1、"区分 |
| 标题字号 | 小二号→三号→四号→小四逐层递减 |
| 脚注 | 宋体小五号，一倍行距 |
| 长引文 | 超三行缩格排版，用仿宋体 |
| 页码 | 记得插入 |

## 第七阶段：学术伦理

### 7.1 调查伦理
- 知情原则：告知身份和目的（公开场合观察例外）
- 同意原则：对方不愿则不必勉强
- 无害原则：可能带来不便的，采取变通措施

### 7.2 写作伦理
- 自称：避免"我们"（除非合作作品）；可用"笔者"或"我"
- 批评和商榷：紧扣学术问题，态度真诚，避免人身攻击

### 7.3 引用伦理（三条红线）
- **绝对禁止抄袭**——照抄或改头换面而不注明出处
- **绝对禁止剽窃**——将他人作品或观点当作自己的发表
- **绝对禁止伪造注释**——没有读过的文献注明"转引自"

### 7.4 投稿伦理
- 一稿一投：被拒或过约定期限再投别家
- 重复发表：原则上不允许，确需再发表须征得双方同意
- 转载：须标示原始出版信息

## 全书方法论速查表

| 阶段 | 核心任务 | 关键产出 | 参考耗时 |
|------|---------|---------|---------|
| 选题 | 发现→提炼→检验→确定 | 核心命题（一句话） | 1—3个月 |
| 文献 | 地毯检索→延伸阅读→整理 | 文献综述 | 1—2个月 |
| 调查 | 观察/访谈/问卷/文献分析 | 一手数据 | 1—3个月 |
| 论证 | 描述→分析→解释→建议 | 论文主体 | 2—6个月 |
| 成文 | 标题→导论→正文→结论→摘要 | 完整初稿 | 1—3个月 |
| 修改 | 措词→句段→结构→排版 | 终稿 | 2—4周 |

## 核心公式

```
好的法学论文 = 
  好的选题（有意义的真问题）+
  充分的文献（穷尽相关研究）+
  严谨的论证（法律解释+逻辑推理）+
  规范的行文（干净/准确/有层次）+
  恪守伦理（原创/诚实/规范）
```
